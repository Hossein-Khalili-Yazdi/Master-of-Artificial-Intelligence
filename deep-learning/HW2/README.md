# рҹ§  Perceptron Logic Gate Simulator

A lightweight Python implementation of a **Single-Layer Perceptron** built from scratch to simulate basic linear logic gates (`AND` and `OR`). 

This project demonstrates how a simple artificial neuron learns linear decision boundaries using weight adjustments and bias updates.

---

## рҹ“Ң Features

* вҡҷпёҸ **From-Scratch Implementation:** Perceptron algorithm built without high-level deep learning frameworks.
* рҹҡӘ **Logic Gate Coverage:** Successfully trains and tests on both **AND** and **OR** logic gates.
* рҹ“Ҳ **Decision Boundary Visualization:** Includes automated Matplotlib plots showing the hyperplanes that separate binary classifications.
* рҹҺҜ **Accuracy Verification:** Console logs and predictive outputs verifying correct logic classification ($100\%$ accuracy).

## рҹӣ пёҸ How it Works
The single-layer perceptron computes the weighted sum of inputs plus a bias term, then applies a step activation function:

$$f(x) = \begin{cases} 1 & \text{if } w_1 x_1 + w_2 x_2 + b \ge 0 \\ 0 & \text{otherwise} \end{cases}$$

During training, weights and bias are updated according to the error rule:$$w_i \leftarrow w_i + \eta (y - \hat{y}) x_i$$

$$w_i \leftarrow w_i + \eta (y - \hat{y}) x_i$$

$$b \leftarrow b + \eta (y - \hat{y})$$

Where:
* $\eta$ is the learning rate.
* $y$ is the target output.
* $\hat{y}$ is the predicted output.
