# 🤖 Machine Learning & Neural Networks Showcase

Welcome to the **Machine Learning** repository! This folder contains a comprehensive collection of hands-on Machine Learning and Deep Learning projects, ranging from fundamental neural network concepts (like Hebbian Learning and Perceptrons) to advanced architectures (CNNs, RNNs, and custom GUIs).

---

## 📌 Projects Overview

| # | Project Name | Type | Key Concepts / Technologies |
|---|---|---|---|
| **01** | **Hebbian Learning for Logic Gates** | Neural Network Basics | Hebbian Theory, Unsupervised Learning, AND/OR Gates |
| **02** | **Perceptron Logic Gate Simulator** | Supervised Learning | Single-Layer Perceptron, Decision Boundaries, Linear Separability |
| **03** | **Multi-Class Perceptron Classification** | Classification | Multi-Layer Perceptron Setup, 4-Class Classification |
| **04** | **Titanic Survival Classifier** | Deep Learning | Binary Classification, Feedforward Neural Networks (FNN), Data Preprocessing |
| **05** | **Auto-MPG Regression** | Deep Learning | Regression Models, Continuous Prediction, Feature Scaling |
| **06** | **RNN Sequence Predictor GUI** | Sequential Models / GUI | SimpleRNN, LSTM, GRU Comparison, Interactive Desktop Interface |
| **07-08** | **CNN Model Evaluation Workstation** | Computer Vision | Convolutional Neural Networks, Performance Benchmarking, Metrics Visualization |

---

## 🚀 Key Highlights & Concepts Covered

* **Fundamental Neural Networks:**
  * **Hebbian Learning:** Demonstrating unsupervised learning principles where connections between neurons strengthen when activated simultaneously.
  * **Perceptrons:** Simulating foundational single-layer architectures to solve classic linearly separable problems (AND & OR gates) and scaling up to multi-class classification problems.

* **Tabular Data & Deep Learning:**
  * **Classification (Titanic Dataset):** Preprocessing categorical/missing data and training a Deep Neural Network to predict passenger survival.
  * **Regression (Auto-MPG Dataset):** Building dense networks to accurately estimate vehicle fuel efficiency based on automobile features.

* **Sequence Modeling & Interactive Tools:**
  * **RNN Benchmarking GUI:** An interactive application designed to visualize and evaluate performance differences between standard **SimpleRNN**, **LSTM** (Long Short-Term Memory), and **GRU** (Gated Recurrent Unit) architectures on sequential data.

* **Computer Vision:**
  * **CNN Evaluation Workstation:** A full setup for benchmarking Convolutional Neural Network architectures, evaluating accuracy, precision, recall, and visualizing training dynamics.

## 🛠️ Tech Stack & Prerequisites

* **Language:** Python 3.x
* **Core Libraries:**
  * Data Processing: `pandas`, `numpy`
  * Machine Learning & Deep Learning: `scikit-learn`, `tensorflow` / `keras` (or `torch`)
  * Visualization: `matplotlib`, `seaborn`
  * GUI Development: `tkinter` / `PyQt` (for the RNN GUI)
